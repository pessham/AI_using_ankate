# AI活用診断ツール - 15問フルスコア版ガイド

このリポジトリには、Googleフォームで収集した回答を1クリックで可視化し、経営陣に「今すぐ研修を導入すべき理由」を伝えるためのツール一式が含まれています。最新バージョンでは推奨15問セットを採用し、**意欲（X）・スキル（Y）・リスク（Z）・総合レベル**の4指標を同時に算出します。

---

## 主な構成物
| ファイル | 役割 |
| --- | --- |
| `ai-diagnosis-tool.html` | React + Recharts で構築した診断レポート（ブラウザで開くだけ） |
| `google-form-template.md` | 15問アンケートとスコアリング設計のテンプレート |
| `sales-presentation.md` | 商談・研修提案用トークスクリプト |
| `PROJECT-SUMMARY.md` | プロジェクトの完成サマリー |

---

## 機能ハイライト
- ✅ **3軸スコア**: 意欲・スキル・リスクを0〜5点で定量化し、業界平均と比較するレーダーチャートを生成
- ✅ **4象限タイプ**: AIリーダー / AI挑戦者 / AI慎重派 / AI未着手を自動分類し、社員構成と訴求ストーリーを生成
- ✅ **リスク判定**: Z軸スコアと危険回答率から「危険 / 警告 / 安全」を判定し、ガバナンス研修CTAを表示
- ✅ **部署別ダッシュボード**: 部署ごとの意欲・スキル・リスクを棒グラフ化し、重点部署を特定
- ✅ **CTA・営業導線**: Googleスライドの埋め込み、助成金説明、30分無料相談（spirincリンク）をワンクリック配置

## セットアップ手順

### 1. アンケートを作る
1. `google-form-template.md` を見ながら Googleフォームを作成
2. 全質問を必須化し、選択肢の文言をテンプレートと揃える（スコアリングが正しく機能）
3. フォームの「回答」タブから「スプレッドシートを作成」し、回答を紐づけ

### 2. スプレッドシートを整える
1. 各質問列の右側にスコア列（任意）と X/Y/Z 列を追加
2. テンプレート記載の `SWITCH / VLOOKUP / LET` 式をコピペして点数化
3. 完成例：
   - `X軸スコア (意欲)` = Q1, Q2, Q3, Q14 の平均
   - `Y軸スコア (スキル)` = Q4, Q5, Q6, Q7, Q8 の平均
   - `Z軸スコア (リスク)` = Q9, Q10, Q11, Q12, Q13 の平均
4. 共有設定を「リンクを知っている全員が閲覧可能」に変更し、URLまたはIDをコピー

> もしスコア列を作らなくても、`ai-diagnosis-tool.html` 側が15問の生データから自動計算します。列名さえ一致していればOKです。

### 3. レポートを開く
1. `ai-diagnosis-tool.html` をブラウザで開く（ローカルファイルでも可）
2. フォームでコピーしたスプレッドシートURLまたはIDを入力
3. 「診断を開始する」を押すと、
   - 総合レベル/危険度コメント
   - 3軸レーダーチャート（自社 vs 業界平均）
   - 4象限カード + 散布図
   - リスク指標と具体的な危険回答率
   - 部署別棒グラフ
   - CTA（守り/攻め/部署別） と Googleスライド埋め込み
   が自動生成されます。

## 使い方のコツ
- **データ更新**: スプレッドシートに新しい回答が追記されたら、ブラウザ側で再読み込みするだけで最新結果になります。
- **プレゼンモード**: ブラウザをプロジェクターに映し、リアルタイムでグラフが描かれる様子を見せると臨場感が出ます。
- **シナリオ表示**: レポート本文のメッセージ（AI挑戦者が◯% 等）は自動で変わるので、そのまま読み上げるだけでコンサル提案が成立します。

## カスタマイズ方法

### デザイン
- Tailwindのユーティリティクラスで配色を変更（例：`.accent-bg` の色を差し替え）
- フォントを変えたい場合は `<style>` 内の Google Fonts を置き換え

### データロジック
- `ai-diagnosis-tool.html` の `QUESTION_SCORES` や `INDUSTRY_BENCHMARK` を編集すれば独自ロジックに変更可能
- 4象限のしきい値を変える場合は `determineQuadrant` 内の `threshold = 3.5` を調整

### CTA・外部リンク
- Googleスライド埋め込みURLは `GOOGLE_SLIDES_EMBED_URL`
- 無料相談URLは `FREE_CONSULTATION_URL`
- それぞれ HTML の冒頭に定数として定義しているので差し替えるだけで反映されます

## 拡張のヒント
1. **PDFエクスポート**: `jspdf` や `html2canvas` を読み込めばレポートをそのままPDF化可能
2. **データベース連携**: Googleスプレッドシートの代わりに自社APIを呼ぶ場合は `fetchSheetData` を差し替え
3. **多言語対応**: 文言を定義ファイルに切り出し、 `Intl` や i18next を使えば英語版も容易に実装できます

## トラブルシューティング
| 症状 | 原因 | 対処 |
| --- | --- | --- |
| 「データ取得エラー」 | スプレッドシートが非公開 | 共有設定を「リンクを知っている全員が閲覧可能」に |
| レーダーチャートが空 | Q1〜Q15 の列名がテンプレと異なる | 列名を見直す / 冒頭と末尾の空白を削除 |
| 4象限散布図が片側に固まる | X/Y スコアが0扱い | 回答が空欄になっていないか確認し、スコア式を見直す |
| CTAリンクが違う | ファイル内の `FREE_CONSULTATION_URL` を修正 | ご自身の予約導線に差し替え |
