# Googleフォーム アンケート質問テンプレート

このテンプレートを使用して、Googleフォームでアンケートを作成してください。

---

## 📋 基本設定

**フォームタイトル**: AI活用状況アンケート  
**説明文**:
```
このアンケートは、社内のAI活用状況を把握し、より効果的な支援体制を整えるために実施します。
所要時間: 約3分
回答内容は統計的に処理され、個人が特定されることはありません。
```

## 🔢 スコア計算（スプレッドシート側で実施）

Googleフォームの回答を受け取ったスプレッドシートに、以下の列を追加してください:

### I列: Q1.頻度スコア
```
=IF(C2="毎日利用",5,IF(C2="週に数回",4,IF(C2="月に数回",2,IF(C2="ほとんど利用していない",1,0))))
```

### J列: Q2.マインドスコア
```
=IF(D2="大きなチャンス",5,IF(D2="ややチャンス",4,IF(D2="どちらでもない",3,IF(D2="やや脅威",2,1))))
```

### K列: Q3.指示スコア
```
=IF(E2="自信がある",5,IF(E2="少しある",3,IF(E2="あまりない",2,1)))
```

### L列: Q4.見抜くスコア
```
=IF(F2="見抜ける",5,IF(F2="多分見抜ける",3,IF(F2="あまり自信がない",1,0)))
```

### M列: X軸スコア（意欲）
```
=(I2+J2)/2
```
※ Q1頻度スコア + Q2マインドスコアの平均

### N列: Y軸スコア（スキル）
```
=(K2+L2)/2
```
※ Q3指示スコア + Q4見抜くスコアの平均

## 🎯 セットアップ手順

### 1. Googleフォームを作成
1. [Google Forms](https://forms.google.com) にアクセス
2. 「+」で新しいフォームを作成
3. 上記の質問を順番に追加

### 2. スプレッドシートに接続
1. フォーム編集画面で「回答」タブをクリック
2. 「スプレッドシートを作成」をクリック
3. 新しいスプレッドシートを作成

### 3. スコア計算列を追加
1. スプレッドシートを開く
2. I列〜O列に上記の計算式を追加
3. 1行目に列名を入力

### 4. 公開設定
1. スプレッドシートの「共有」ボタンをクリック
2. 「リンクを知っている全員が閲覧可能」に変更
3. リンクをコピー

### 5. 診断ツールで使用
1. `ai-diagnosis-tool.html` を開く
2. コピーしたスプレッドシートのURLを入力
3. 「診断を開始する」をクリック

## 💡 運用のコツ

### 回答率を上げるポイント
1. **期限を設定**: 「〇月〇日までにご回答ください」
2. **所要時間を明記**: 「約3分で完了」
3. **匿名性を強調**: 「個人は特定されません」
4. **経営層からの依頼**: 社長や部長名でアナウンス

### データ品質を保つポイント
1. 全質問を必須にする
2. 社員IDの形式を統一（例: emp_001）
3. 部署名を正確に選択してもらう
4. 定期的に実施して変化を追跡
