# Googleフォーム & スプレッドシート完全テンプレート（15問フル診断）

このテンプレートは、AI活用状況を「意欲・スキル・リスク」の3軸で診断し、経営層に刺さるレポートを作るための**本格版アンケート（15問）**と、集計用Googleスプレッドシートの作り方をまとめたものです。ここで定義した列名とスコアリングを守れば、`ai-diagnosis-tool.html` がそのまま読み取り、ワンクリックでレポート化できます。

---

## セットアップ全体像
1. **Googleフォーム**でアンケートを作成（以下の質問を順番に追加）
2. **Googleスプレッドシート**に回答を連携し、スコア列を追加
3. スプレッドシートを「リンクを知っている全員が閲覧可能」に設定
4. `ai-diagnosis-tool.html` をブラウザで開き、スプレッドシートのURL（またはID）を入力

---

## 質問セット（全15問）

> 所要時間：最短3分、長くても10分。全質問を必須にするのが推奨です。

### 回答者情報
- **Q0. あなたの所属部署をお選びください**（選択式 / 営業部・開発部・マーケ部・管理部・バックオフィス・その他）
- **Q0-2. 役職 or 職種（任意）**（短文 / 例：課長、エンジニア）

### カテゴリ1：AI活用への意識（X軸を構成）
1. **Q1. AIに対するあなたの考えに最も近いものは？**（単一選択）
   - 5点: 自分の仕事を劇的に良くする「チャンス」
   - 4点: 便利な「ツール」
   - 3点: どちらでもない
   - 2点: 使い方を覚えるのが「面倒」
   - 1点: 自分の仕事を奪う「脅威」
2. **Q2. 業務におけるAI（ChatGPT/Copilotなど）の利用頻度は？**（単一選択）
   - 5点: 毎日利用する
   - 4点: 週に数回利用する
   - 3点: 月に数回利用する
   - 2点: 使おうとしたが、やめた
   - 1点: 全く利用しない
3. **Q3. 今後、AIをどれくらい活用したいですか？**（単一選択）
   - 5点: 業務に積極的に活用したい
   - 4点: 機会があれば活用したい
   - 3点: どちらでもない
   - 2点: あまり活用したくない
   - 1点: 全く活用したくない

### カテゴリ2：AI活用の実践（Y軸を構成）
4. **Q4. 現在、主にどのAIツールを利用していますか？**（チェックボックス / 複数選択可）
   - ChatGPT (無料版)
   - ChatGPT (有料版: Plus/Teamなど)
   - Microsoft Copilot (WindowsやEdge)
   - Gemini (Google)
   - Claude
   - 社内独自のAIツール
   - その他（自由記述）
   - 特に利用していない（これを選んだ場合はスコア0）
5. **Q5. AIをどのような業務に活用していますか？**（チェックボックス）
   - 文章作成・編集・要約
   - アイデア出し・ブレインストーミング
   - データ分析・集計
   - 翻訳
   - コーディング・プログラミング支援
   - 画像・デザイン生成
   - その他（自由記述）
   - 活用していない（選択時はスコア0）
6. **Q6. AIに指示（プロンプト）を出す際の行動は？**（単一選択）
   - 5点: 役割や条件、出力形式を具体的に指示し、回答をコントロールしている
   - 3点: 簡単な質問や単語で、ひとまず指示している
   - 1点: 指示の仕方がよくわからず、期待通りの回答が得られない
7. **Q7. AIの活用によって業務効率は向上しましたか？**（単一選択）
   - 5点: 大幅に向上した（半分以上削減）
   - 4点: やや向上した
   - 3点: 少し向上した
   - 2点: ほとんど変わらない
   - 1点: 活用していない / わからない
8. **Q8. AIに関する研修や学習の経験は？**（単一選択）
   - 5点: 会社主催の体系的な研修を受けた
   - 4点: Web記事や動画、書籍などで独学している
   - 3点: 社内の簡単な勉強会や説明会に参加した
   - 1点: 特に学習したことはない

### カテゴリ3：AI利用のリスク（Z軸を構成）
9. **Q9. AIが生成した回答を業務で利用する際の行動は？**（単一選択）
   - 5点: 必ずファクトチェックや内容修正をしている
   - 3点: 重要な内容の時だけ検証している
   - 1点: ほぼ検証せず、そのまま利用することが多い
10. **Q10. AIに社外秘情報を入力することについての考えは？**（単一選択）
    - 5点: 会社のルールを理解し、絶対に入力しない
    - 3点: 入力してはいけないと思うが、何が社外秘か曖昧
    - 1点: 業務効率化のためなら入力しても問題ない / わからない
11. **Q11. AIが生成した成果物の著作権についての理解は？**（単一選択）
    - 5点: 侵害の可能性を理解している
    - 3点: ルール（法律）が曖昧でよくわからない
    - 1点: AIが作ったものなので著作権は問題ないと思う
12. **Q12. AIの回答にバイアスが含まれる可能性を知っていますか？**（単一選択）
    - 5点: 知っており、常に中立性を意識して確認している
    - 3点: 言葉は知っているが、具体的にはわからない
    - 1点: 知らない / AIは中立だと思う

### カテゴリ4：組織としての課題
13. **Q13. 会社（部署）のAI利用ガイドラインは？**（単一選択）
    - 5点: 明文化されたガイドラインがあり、内容を理解している
    - 3点: ガイドラインの存在は知っているが内容はよく知らない
    - 2点: 「機密情報を入れるな」等の口頭注意のみ
    - 1点: 特にルールは無い / わからない
14. **Q14. AIスキルを学ぶ時間についての考えは？**（単一選択）
    - 5点: 必要性を強く感じ、業務時間外でも学習したい
    - 4点: 会社が認める業務時間内であれば学習したい
    - 3点: 必要に迫られたら学習する
    - 1点: 他に優先すべき業務があり、学習する時間はない
15. **Q15. AI活用における最も大きな課題は？**（チェックボックス / 複数選択可）
    - AIを使いこなすスキル・知識が不足している
    - セキュリティや情報漏洩が心配
    - どの業務に使えるかわからない
    - 会社（上司）の理解や方針がない
    - 利用するための時間がない
    - 費用（有料ツール導入など）がかかる
    - 特に課題はない

## スプレッドシート設計

### 列構成（推奨）
| 列 | 項目名 | 備考 |
| --- | --- | --- |
| A | タイムスタンプ | 自動入力 |
| B | 社員ID（任意） | 短文 / 任意で匿名化コード |
| C | 所属部署 | Q0 |
| D-R | Q1〜Q15の回答 | 1列1質問（上記の質問文をそのまま列名に） |
| S | X軸スコア (意欲) | 可視化用（自動計算） |
| T | Y軸スコア (スキル) | 可視化用（自動計算） |
| U | Z軸スコア (リスク) | 可視化用（自動計算） |
| V | リスク判定 | 危険/警告/安全（任意） |
| W | 4象限タイプ | AIリーダー / AI挑戦者 etc.（任意） |

> `ai-diagnosis-tool` は Q1〜Q15 の列名さえあれば自動でスコアを計算します。スプレッドシート側でも上記のスコア列を作っておくと、フォーム編集者だけで状況を把握しやすくなります。

### 単一選択のスコア変換
代表例（Q1）の式：
```google-sheets
=SWITCH(D2,
  "自分の仕事を劇的に良くする「チャンス」",5,
  "便利な「ツール」",4,
  "どちらでもない",3,
  "使い方を覚えるのが「面倒」",2,
  "自分の仕事を奪う「脅威」",1,
  1)
```
他の単一選択も同じ構造で作成できます（SWITCHやIFSを活用）。

### 複数選択のスコア例（Q4/Q5/Q15）
1. `設定`シートにスコア表を作成しておく：
   | ツール | 点数 |
   | --- | --- |
   | ChatGPT (無料版) | 1.5 |
   | ChatGPT (有料版: Plus/Teamなど) | 2 |
   | Microsoft Copilot (WindowsやEdge) | 1.5 |
   | Gemini (Google) | 1.5 |
   | Claude | 2 |
   | 社内独自のAIツール | 2.5 |
   | その他 | 1 |
   | 特に利用していない | 0 |
2. 回答列が `G2` の場合の例：
```google-sheets
=LET(
  raw, TRIM(G2),
  items, FILTER(SPLIT(raw, ","), LEN(SPLIT(raw, ","))),
  score, IF(COUNTIF(items, "特に利用していない"), 0,
            SUMPRODUCT(
              ARRAYFORMULA(VLOOKUP(items, 設定!$A$2:$B$9, 2, FALSE))
            )
         ),
  MIN(5, score)
)
```
3. Q5/Q15も同様に `VLOOKUP` と `MIN(5, …)` で調整します。`活用していない` / `特に課題はない` を選んだら 0 点になるようにします。

### 3軸スコア計算
```google-sheets
# X軸（意欲）：Q1, Q2, Q3, Q14 の平均
=ROUND(AVERAGE(Q1スコア:Q14スコア), 1)

# Y軸（スキル）：Q4, Q5, Q6, Q7, Q8 の平均
=ROUND(AVERAGE(Q4スコア:Q8スコア), 1)

# Z軸（リスク）：Q9, Q10, Q11, Q12, Q13 の平均
=ROUND(AVERAGE(Q9スコア:Q13スコア), 1)
```
※ 実際は `AVERAGE({範囲})` を各列参照で作成してください。`ROUND(…,1)` で小数1桁に整えます。

### 4象限タイプ & リスク判定（任意）
```google-sheets
=IFS(
  AND($S2>=3.5, $T2>=3.5), "AIリーダー",
  AND($S2>=3.5, $T2<3.5), "AI挑戦者",
  AND($S2<3.5, $T2>=3.5), "AI慎重派",
  TRUE, "AI未着手"
)

=IFS(
  $U2<2.5, "危険",
  $U2<3.5, "警告",
  TRUE, "安全"
)
```

## 公開設定
1. スプレッドシート右上の「共有」→「一般的なアクセス」を「リンクを知っている全員が閲覧可能」に変更
2. URLは `https://docs.google.com/spreadsheets/d/スプレッドシートID/edit#gid=0` の形式
3. `ai-diagnosis-tool.html` では「URL」でも「ID」でも入力可能（IDのみを抜き出してもOK）

---

## 運用ヒント
- 期日と所要時間（3〜10分）を必ずアナウンスし回答率を高めます
- 経営層や部門長からの依頼メールにして「公式調査」であることを明示
- 初回は全社一斉、以降は部署単位で四半期ごとに定点観測すると改善が見えます
- 回答後すぐにスプレッドシートへ流れ込むので、`ai-diagnosis-tool.html` を開いたままリロードすればリアルタイムに結果をプレゼンできます
